Atividades


Atividade 1

O aplicativo permite aos usuários explorar os números pares até um determinado valor, fornecendo uma soma dos números pares e uma lista para referência.
Atividade 2

O aplicativo "Conversor de Temperatura" permite aos usuários converter facilmente uma temperatura de Fahrenheit para Celsius ou de Celsius para Fahrenheit.
Atividade 3

O aplicativo "Calculadora de Vendas" permite aos usuários calcular o valor total de uma venda com base nos produtos, seus valores unitários e quantidades.
Atividade 4

O aplicativo permite aos usuários calcular o gasto de combustível com base na quilometragem percorrida e no valor do combustível.
Atividade 5

O aplicativo permite aos usuários calcular o custo de assentar piso em um cômodo com base no comprimento e largura fornecidos.
Atividade 6

O aplicativo permite aos usuários calcular seu peso ideal com base na altura e no sexo fornecidos.
Atividade 7

O aplicativo permite que os usuários adicionem valores, exibam os valores adicionados e encontrem o maior e o menor valor entre os valores adicionados.
Jogo da velha

o aplicativo de jogo da velha apresenta uma interface simples e funcional, permitindo que os usuários joguem o jogo clássico em um ambiente web interativo.
